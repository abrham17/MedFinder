# MedFinder Ethiopia - Requirements

## Server Requirements
- PHP 7.4 or higher
- MySQL 5.7+ or MariaDB 10.3+
- Apache 2.4+ with mod_rewrite
- 256MB RAM minimum
- 100MB disk space

## PHP Extensions
- mysqli
- session
- json
- mbstring

## Browser Support
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## Development Tools
- Git
- Text editor (VS Code recommended)
- phpMyAdmin (optional, for DB management)
